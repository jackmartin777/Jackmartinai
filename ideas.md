# Design Ideas — Jack Martin Portfolio

## Context
Portfolio + CV website for a digital strategist & AI consultant. Must match the existing CV design: dark Oxford blue background, burnt orange accents, brain/heart watermark image, Baloo 2 + Inter fonts.

## Idea 1: "Neural Terminal"

**Design Movement:** Cyberpunk-meets-academic — inspired by terminal interfaces and scientific dashboards.

**Core Principles:** Monospace code aesthetics, data-driven layouts, command-line inspired navigation, glowing accent lines.

**Color Philosophy:** Oxford blue (#0b1224) as the void, burnt orange (#e8833a) as the signal — like a terminal cursor blinking in the dark. All information is "data" being rendered.

**Layout Paradigm:** Full-screen sections with terminal-style borders. Content appears as if being "typed out." Portfolio cards are rendered like file system entries.

**Signature Elements:** Blinking cursor animations, code-block styled project cards, ASCII-art section dividers.

**Interaction Philosophy:** Hover states reveal data like tooltips in an IDE. Scroll triggers "render" animations.

**Animation:** Typewriter text reveals, fade-in from bottom with stagger, glowing border pulses on hover.

**Typography System:** JetBrains Mono for headings and labels, Inter for body text. Monospace creates the "tech" feel.

## Idea 2: "Midnight Architect"

**Design Movement:** Dark luxury minimalism — inspired by high-end product pages (Apple, Porsche) but with warm accents.

**Core Principles:** Generous whitespace on dark canvas, large typography that breathes, content hierarchy through scale contrast, subtle depth through layered cards.

**Color Philosophy:** Deep Oxford blue as a canvas of calm authority. Burnt orange used sparingly — only for CTAs, active states, and key metrics. The restraint makes the orange feel precious and intentional.

**Layout Paradigm:** Asymmetric sections with large left-aligned headings. Content blocks float on subtle card layers. Portfolio uses a masonry-inspired grid with hover reveals.

**Signature Elements:** The brain/heart watermark as a persistent subtle background, oversized section numbers (01, 02, 03), thin orange accent lines as dividers.

**Interaction Philosophy:** Smooth parallax scrolling, cards that lift on hover with shadow depth, magnetic cursor effects on CTAs.

**Animation:** Scroll-triggered fade-up with stagger delays, smooth section transitions, subtle scale on hover, text gradient shimmer on the name.

**Typography System:** Baloo 2 (800 weight) for display headings at massive scale, Inter (300-600) for everything else. The contrast between playful-bold and clean-light creates sophistication.

## Idea 3: "Constellation Map"

**Design Movement:** Data visualization aesthetic — inspired by star maps, network graphs, and scientific posters.

**Core Principles:** Information as visual art, connecting dots between disciplines, node-and-edge metaphors for skills/projects, observatory-like observation of work.

**Color Philosophy:** Oxford blue as deep space. Burnt orange as stars and connection points. White as distant light. The CV becomes a map of a professional constellation.

**Layout Paradigm:** Radial layouts for skills, timeline as a vertical constellation path, projects as orbiting nodes. Non-traditional grid that feels like exploring a map.

**Signature Elements:** Dot-and-line connection graphics between sections, radial skill visualizations, orbital project cards, star-field particle background.

**Interaction Philosophy:** Hovering on nodes reveals connections. Scrolling feels like zooming through space. Projects orbit and settle into view.

**Animation:** Particle drift in background, orbital hover on project cards, connection lines that draw on scroll, pulsing nodes.

**Typography System:** Baloo 2 for constellation labels (bold, uppercase), Inter thin for descriptions. Small caps for categories.
</text>

## Selected: Idea 2 — "Midnight Architect"

This best matches the existing CV aesthetic (dark Oxford blue, burnt orange accents, Baloo 2 + Inter) and creates a premium, professional portfolio that positions Jack as a high-end consultant. The restraint and luxury feel aligns with his multidisciplinary authority.
